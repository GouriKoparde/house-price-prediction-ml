# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices based on features like area, bedrooms, bathrooms, age, and parking using regression models.

## 🎯 Problem Statement

Real estate pricing is complex and depends on multiple factors. This project builds a machine learning model to estimate property prices accurately.

## 💼 Industry Relevance

* Used by real estate platforms (housing portals)
* Helps banks in loan valuation
* Assists buyers & sellers in price estimation
* Supports investors in decision-making

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* Joblib

## 🤖 Models Used

* Linear Regression
* Random Forest Regressor

## 📊 Features Used

* Area (sq ft)
* Bedrooms
* Bathrooms
* Age of property
* Parking availability

## 📈 Model Evaluation

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score
